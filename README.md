## 🏥 Medicare Admin Dashboard

A modern, scalable healthcare administration dashboard built with Next.js 16 and React 18. Designed for performance, maintainability, and a clean user experience, this system provides a solid foundation for managing healthcare operations efficiently.

## 🚀 Features

- ⚡ High-performance frontend with Next.js (App Router ready)
- 🎯 Centralized state management using Redux Toolkit
- 🧩 Scalable architecture for large applications
- 📊 Interactive charts and analytics with Recharts
- 📝 Efficient form handling using React Hook Form
- 🔔 Real-time notifications via React Hot Toast
- 🎨 Clean, responsive UI with Tailwind CSS
- 📅 Advanced date handling using date-fns
- 🧠 Fully typed with TypeScript for better reliability
